# Coc Viec Tu Xa

Chào mừng bạn đến với Coc Viec Tu Xa, một ứng dụng tuyệt vời dành cho những ai đang tìm kiếm cơ hội làm việc từ xa tại Việt Nam.

## Mục tiêu
Coc Viec Tu Xa giúp kết nối những người tìm việc làm với các công ty đang có nhu cầu tuyển dụng nhân sự làm việc từ xa. Chúng tôi muốn tạo ra một nền tảng thân thiện và dễ sử dụng, nơi người lao động có thể tìm kiếm, ứng tuyển và quản lý hồ sơ của mình một cách dễ dàng.

## Tính năng chính
- **Tìm kiếm việc làm:** Người dùng có thể tìm kiếm việc làm từ xa theo ngành nghề, kỹ năng, và mức lương mong muốn.
- **Gửi hồ sơ:** Người dùng có thể tạo và gửi hồ sơ trực tuyến.
- **Đánh giá công ty:** Người lao động có thể đánh giá và ghi nhận trải nghiệm làm việc với các công ty.
- **Thông báo việc làm mới:** Nhận thông báo hàng tuần về các vị trí việc làm mới.

## Cài đặt
```
# Clone the repo
git clone https://github.com/<your-username>/coc-viec-tu-xa.git

# Install dependencies
cd coc-viec-tu-xa
npm install
```

## Công nghệ sử dụng
- React để xây dựng giao diện người dùng
- Node.js cho backend
- MongoDB để lưu trữ dữ liệu
